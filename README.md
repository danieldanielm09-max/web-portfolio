# Daniel Mathew — Flet Web Portfolio

## Setup

```bash
pip install flet
```

## Run locally

```bash
python main.py
```

Opens at http://localhost:8080

## Deploy (Flet Cloud / any server)

```bash
flet publish main.py
```

## Video files

Place the two video files in the same folder as main.py:

- `project_2__1_.mp4`
- `project_3.mp4`

Then update the `video_path` arguments in the blog section if the paths differ.
